# claude-session-kit

**Persistent Claude Agent SDK sessions (streaming-input mode): keep the CLI subprocess alive across turns for 8x prompt-cache efficiency, with per-turn results, interrupt/setModel, cache stats, and a per-project session pool.** 零依賴、框架無關。

## 安裝

```bash
npm i github:Jeffrey0117/claude-session-kit   # 或直接複製 index.js
```

## 用法

```js
const create = require('claude-session-kit');
const x = create({ /* opts */ });
```
